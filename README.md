# TicoMarket - Sistema de Gestión de Ventas

Proyecto desarrollado en Java para la gestión de un sistema de ventas en línea, utilizando estructuras de datos dinámicas para el control del inventario y los pedidos realizados.

## Funcionalidades

- Agregar productos al inventario
- Mostrar inventario en orden alfabético
- Registrar clientes y llenar su carrito de compras
- Cola de atención con prioridad por tipo de cliente
- Generación de factura con costo total al atender un cliente
- Menú interactivo con arte ASCII

## Estructuras de datos utilizadas

- Lista enlazada (carrito de cada cliente)
- Árbol binario de búsqueda (inventario de la tienda)
- Cola de prioridad (atención de clientes)

## Tecnologías utilizadas

- Java
- Estructuras de Datos
- Programación Orientada a Objetos
