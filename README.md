# Sistema de Inventario - Lista Enlazada

Proyecto desarrollado en Java que implementa una lista enlazada para la gestión de productos en un supermercado.

## Funcionalidades

- Agregar productos
- Modificar productos
- Eliminar productos
- Agregar imagen a productos
- Reporte de costos
- Menú interactivo con arte ASCII

## Tecnologías utilizadas

- Java
- Estructuras de Datos (Lista Enlazada)
- Programación Orientada a Objetos
